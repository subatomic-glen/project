# project
DS Toolkit Assignment 6

This project was created for Assignment 6 of the Data Science Toolkit course. The project requires students to practice the following:

- Create a repository and branches on github
- Clone a repository to local
- Fork a github repository
- Add content to a repository
- Organize a project per best practices
- Name files according to best practices
- Commit and push changes from local to remote and vice-versa
- Create a virtual environment
- Create and manually modify a yaml file for project collaborators
- Create a pull request and merge branches
